# AI-SKILLS

Denne mappen er mellomstasjonen mellom prosjektet og felles AI-SKILLS-metodebase.

Ikke alt som oppdages i et prosjekt skal rett inn i felles AI-SKILLS. Bruk denne mappen til å samle, sortere og modne forslag.

## Filer

- `kandidater.md`: konkrete forslag som virker relevante.
- `backlog.md`: forslag som bør vurderes senere eller testes mer.

## Bruksregel

Når et prosjekt er ferdig:

1. Skriv reflection.
2. Identifiser gjenbrukbar læring.
3. Legg modne forslag i `kandidater.md`.
4. Legg usikre forslag i `backlog.md`.
5. Ikke tilbakefør forslag som bare er lokale for prosjektet.
