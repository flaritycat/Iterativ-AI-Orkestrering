# Prompt 07: AI-SKILLS-tilbakeføring

## Formål

Bruk denne når prosjektlæring skal gjøres om til konkrete forslag for felles AI-SKILLS.

## Prompt

```text
Du skal vurdere hvilke erfaringer fra dette prosjektet som bør tilbakeføres til AI-SKILLS.

Ikke foreslå alt. Foreslå bare det som har tydelig gjenbruksverdi.

For hvert forslag skal du bruke denne strukturen:

1. Tittel
2. Type endring
   - ny prompt
   - forbedret prompt
   - ny regel
   - endret regel
   - ny mal
   - forbedret arbeidsflyt
   - erfaringsnotat
3. Hva forslaget går ut på
4. Hvilket prosjektbehov det bygger på
5. Hvorfor dette kan brukes i andre prosjekter
6. Hvor i AI-SKILLS det kan høre hjemme
7. Hvordan forslaget bør testes
8. Prioritet

Avslutt med:

- forslag som bør løftes nå
- forslag som bør stå i backlog
- forslag som ikke bør tilbakeføres

Prosjektmateriale:
[LIM INN REFLECTION, PROSJEKTCASE ELLER RELEVANTE NOTATER HER]
```
