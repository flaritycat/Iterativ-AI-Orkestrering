# Prompt 08: Presentasjonsutvikling

## Formål

Bruk denne når metoden skal gjøres om til en presentasjon.

## Prompt

```text
Du skal hjelpe meg å lage en presentasjon om Iterativ AI-orkestrering.

Presentasjonen skal forklare metoden praktisk, ikke akademisk.

Målgruppe:
[BESKRIV MÅLGRUPPE]

Formål:
[BESKRIV HVA PRESENTASJONEN SKAL OPPNÅ]

Bruk denne strukturen som utgangspunkt:

1. Tittel og hovedbudskap
2. Problemet med tilfeldig AI-bruk
3. Hva Iterativ AI-orkestrering er
4. De fire hovedkomponentene: ChatGPT, Codex, GitHub og AI-SKILLS
5. Intervjurunder som ekstraksjonsmotor
6. Prompt-destillering og ny operativ chat
7. Codex-handoff og GitHub som prosjektbase
8. Reflection og AI-SKILLS-tilbakeføring
9. Prosjektcase som leveranse og læringsnode
10. Gevinster og risikoer
11. Hvordan starte i praksis

For hver slide skal du lage:

- slide-tittel
- hovedpoeng
- forslag til visualisering
- kort talestøtte

Hold teksten kort. Bruk visualiseringer til å forklare flyt og sammenheng.
```
