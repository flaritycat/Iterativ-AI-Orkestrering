# Visuals

Denne mappen samler visualiseringsnotater for metoden.

Bruk visualiseringer til å forklare sammenheng, ikke til pynt.

Prioriter figurer som hjelper en ny leser å forstå:

- arbeidsflyten
- rollefordelingen
- læringssløyfen
- hvordan et prosjekt blir både leveranse og læringsnode

Se `visualiseringsliste.md` for foreslåtte figurer.
