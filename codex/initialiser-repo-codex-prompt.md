# Codex-prompt: initialiser metode-repo

```text
Du skal initialisere et GitHub-repository som metodebase for Iterativ AI-orkestrering.

Repoet skal forklare hvordan ChatGPT, Codex, GitHub og AI-SKILLS brukes sammen i en iterativ arbeidsflyt der hvert prosjekt både:

1. produserer en konkret leveranse
2. skaper læring
3. forbedrer den felles metoden som neste prosjekt bygger videre på

Bruk underliggende metodedokument i `source-documents/` som hovedkilde.

Opprett eller oppdater:

- README
- MANIFEST
- CHANGELOG
- docs
- prompts
- templates
- examples
- codex
- ai-skills
- visuals

Skriv på norsk.

Hold alt praktisk, konkret og direkte anvendbart.

Ikke lag tilfeldig dokumentarkiv. Repoet skal være en strukturert metodebase.

Når du er ferdig, skal du:

- oppsummere filene som ble opprettet eller endret
- verifisere at ønsket struktur finnes
- peke på neste praktiske forbedring
```
