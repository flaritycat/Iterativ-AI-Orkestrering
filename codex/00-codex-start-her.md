# Codex: start her

Dette repoet er en metodebase for Iterativ AI-orkestrering.

Når Codex arbeider her, er målet vanligvis ett av disse:

- forbedre dokumentasjon
- legge til eller forbedre promptmaler
- oppdatere prosjektmaler
- dokumentere et prosjektcase
- identifisere AI-SKILLS-kandidater
- gjøre repoet enklere å forstå for nye lesere

## Arbeidsregler

- Les `README.md` og `MANIFEST.md` før større endringer.
- Bevar enkel struktur.
- Skriv på norsk.
- Gjør dokumentasjon praktisk og direkte anvendbar.
- Ikke legg inn akademisk fylltekst.
- Ikke legg til nye mapper uten tydelig behov.
- Oppdater lenker når filer flyttes eller får nytt navn.
- Hvis arbeidet gir gjenbrukbar læring, oppdater `ai-skills/kandidater.md` eller `ai-skills/backlog.md`.

## Vanlig arbeidsflyt

1. Sjekk status og filstruktur.
2. Les relevante docs, prompts eller templates.
3. Gjør små, konsistente endringer.
4. Verifiser lenker og filnavn.
5. Oppsummer hva som ble endret.
6. Foreslå neste steg.
