# Codex-prompt: vedlikehold metode-repo

```text
Du skal vedlikeholde repositoryet for Iterativ AI-orkestrering.

Start med å lese:

- README.md
- MANIFEST.md
- codex/00-codex-start-her.md
- relevante filer for oppgaven

Oppgaven er:

[BESKRIV OPPGAVE]

Regler:

- Bevar repoets enkle struktur.
- Skriv på norsk.
- Gjør innhold praktisk og gjenbrukbart.
- Ikke legg til abstraksjoner uten tydelig nytte.
- Oppdater lenker og manifest hvis strukturen endres.
- Hvis du oppdager læring som bør tilbakeføres til AI-SKILLS, oppdater `ai-skills/kandidater.md` eller `ai-skills/backlog.md`.
- Ikke overskriv prosjektspesifikk læring uten å bevare intensjonen.

Slutt med:

- hva som ble endret
- hvordan det ble verifisert
- åpne spørsmål
- anbefalt neste steg
```
