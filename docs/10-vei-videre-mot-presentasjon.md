# Vei videre mot presentasjon

Metoden kan gjøres om til en kort presentasjon når repoet er stabilt nok.

Presentasjonen bør ikke prøve å forklare alle detaljer. Den bør vise modellen, arbeidsflyten og hvorfor læringssløyfen er nyttig.

Se `visuals/visualiseringsliste.md` for mer konkret oversikt.

## Presentasjonsregel

Bruk ett tydelig poeng per slide.

Metoden er sterkest når den vises som praktisk arbeidsflyt, ikke som teori.
